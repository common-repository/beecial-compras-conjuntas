Gracias a Beecial podrás integrar en tu tienda online las compras conjuntas o compras en grupos para ofrecer a tus usuarios las mejores ofertas y descuentos.

== Descripción ==

Reduce los costes de adquisición de tus clientes.

Al realizar ofertas en forma de conjunta podrás mejorar tus márgenes y reducir tus costes de  adquisición de clientes, ya que serán los propios usuarios los que recomienden tu producto para conseguir la oferta antes.
El proceso de compra es muy sencillo para el usuario y también la forma de compartir las ofertas con otros posibles clientes.

== Instalación ==

Instale el plugin y rellene la configuración del plugin con los datos obtenidos en Beecial
